RandomAlphanumeric(int, bool)

Creates a randomized alphanumeric string.
If allowRepeats is true, characters can be repeated, and the result will always be the length of length.
If length is greater than 62, and allowRepeats is false, the string will only be 62 characters in length, as there is only that many characters to choose from without duplicating.
